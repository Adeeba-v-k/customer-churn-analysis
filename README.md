# Customer Churn Analysis

## Project Overview
This project analyzes telecom customer data to understand the factors that influence customer churn. The objective is to identify patterns in customer behavior and provide insights that may help businesses improve customer retention. The project includes data cleaning, exploratory data analysis, SQL-based business queries, and an interactive Power BI dashboard.

## Dataset
The dataset contains **7,043 customer records** with **21 features**, including customer demographics, service details, billing information, and churn status.

Some important features in the dataset include:
- Gender
- Senior Citizen
- Tenure
- Contract Type
- Internet Service
- Payment Method
- Monthly Charges
- Total Charges
- Churn Status

## Tools Used
- **Python** – Data cleaning and Exploratory Data Analysis (Pandas, Matplotlib, Seaborn)
- **SQL** – Business analysis queries
- **Power BI** – Interactive dashboard and data visualization

## Project Workflow
1. Data understanding and exploration  
2. Data cleaning and preprocessing using Python  
3. Exploratory Data Analysis (EDA)  
4. SQL queries for business analysis  
5. Power BI dashboard creation  
6. Extracting insights from the data  

## Key Insights
- Customers with **month-to-month contracts** show higher churn.
- Customers with **higher monthly charges** tend to churn more frequently.
- **Fiber optic internet users** show relatively higher churn compared to other services.
- **New customers (low tenure)** are more likely to leave early.
- Customers using **electronic check payment methods** show higher churn.

## Dashboard
The Power BI dashboard provides an interactive view of customer churn patterns, including:

- KPI cards showing **Total Customers, Churn Customers, and Churn Rate**
- Churn analysis by **Contract Type**
- Churn analysis by **Internet Service**
- Churn analysis by **Payment Method**
- Churn analysis based on **Customer Tenure**
- Interactive filters for better data exploration
